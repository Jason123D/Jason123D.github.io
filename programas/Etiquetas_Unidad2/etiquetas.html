102 Etiquetas HTML

Nombre de la Etiqueta Descripción Ejemplo de Uso Notas Adicionales
<html> Define el inicio de un documento HTML y contiene todos los demás elementos HTML. 
<html>
    <head>
        <title>Título de la página</title>
    </head>
    <body>
        <p>Este es un párrafo.</p>
    </body>
</html>
 Es la etiqueta raíz de cualquier documento HTML.
<head> Contiene metadatos/información sobre el documento, como el título, enlaces a hojas de estilo, scripts, etc. 
<head>
    <title>Título de la página</title>
    <link rel="stylesheet" href="styles.css">
</head>
 Los elementos dentro de <head> no se muestran directamente en la página.
<title> Define el título del documento, que se muestra en la pestaña del navegador. 
<title>Mi primera página web</title>
 Es importante para el SEO y la accesibilidad.
<body> Contiene el contenido principal del documento HTML, como texto, imágenes, enlaces, etc. 
<body>
    <h1>Bienvenido a mi página</h1>
    <p>Este es un párrafo.</p>
</body>
 Todo el contenido visible de la página debe estar dentro de <body>.
<h1> a <h6> Representan encabezados o títulos, donde <h1> es el más importante y <h6> el menos importante. 
<h1>Título principal</h1>
<h2>Subtítulo</h2>
<h3>Sub-subtítulo</h3>
 Usa solo un <h1> por página para mejorar el SEO.
<p> Define un párrafo de texto. 
<p>Este es un párrafo de ejemplo.</p>
 Los párrafos se separan automáticamente con un margen.
<a> Define un hipervínculo o enlace a otra página o recurso. 
<a href="https://www.ejemplo.com">Visita Ejemplo.com</a>
 El atributo href especifica la URL del enlace.
<img> Inserta una imagen en la página. 
<img src="imagen.jpg" alt="Descripción de la imagen">
 El atributo alt es importante para la accesibilidad.
<ul> Define una lista desordenada (con viñetas). 
<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
</ul>
 Usa <li> para cada elemento de la lista.
<ol> Define una lista ordenada (con números). 
<ol>
    <li>Primer elemento</li>
    <li>Segundo elemento</li>
</ol>
 Similar a <ul>, pero los elementos están numerados.
<li> Define un elemento de una lista, ya sea ordenada (<ol>) o desordenada (<ul>). 
<ul>
                    <li>Elemento 1</li>
                    <li>Elemento 2</li>
                </ul>
 Debe estar dentro de <ul> o <ol>.
<div> Define una sección o contenedor genérico en un documento HTML. 
<div>
                    <p>Este es un párrafo dentro de un div.</p>
                </div>
 Se usa comúnmente para agrupar elementos y aplicar estilos.
<span> Define una sección en línea dentro de un texto. 
<p>Este es un <span style="color: red;">texto en rojo</span>.</p>
 Útil para aplicar estilos a partes específicas de un texto.
<table> Define una tabla en HTML. 
<table>
                    <tr>
                        <th>Nombre</th>
                        <th>Edad</th>
                    </tr>
                    <tr>
                        <td>Juan</td>
                        <td>25</td>
                    </tr>
                </table>
 Se usa junto con <tr>, <th>, y <td> para crear filas y celdas.
<tr> Define una fila en una tabla. 
<tr>
                    <td>Dato 1</td>
                    <td>Dato 2</td>
                </tr>
 Debe estar dentro de <table>.
<td> Define una celda de datos en una tabla. 
<td>Contenido de la celda</td>
 Debe estar dentro de <tr>.
<th> Define una celda de encabezado en una tabla. 
<th>Nombre</th>
 Se usa para resaltar los encabezados de las columnas o filas.
<form> Define un formulario para recopilar datos del usuario. 
<form action="/submit" method="post">
                    <input type="text" name="nombre">
                    <button type="submit">Enviar</button>
                </form>
 Se usa junto con elementos como <input>, <button>, etc.
<input> Define un campo de entrada para que el usuario ingrese datos. 
<input type="text" name="nombre" placeholder="Ingresa tu nombre">
 El atributo type define el tipo de entrada (texto, contraseña, etc.).
<button> Define un botón en el que el usuario puede hacer clic. 
<button type="submit">Enviar</button>
 Puede usarse dentro de formularios o de manera independiente.
<label> Define una etiqueta para un elemento de formulario. 
<label for="nombre">Nombre:</label>
                <input type="text" id="nombre" name="nombre">
 Mejora la accesibilidad al asociar texto con un campo de entrada.
<textarea> Define un área de texto multilínea para que el usuario ingrese datos. 
<textarea name="comentario" rows="4" cols="50"></textarea>
 Útil para comentarios o descripciones largas.
<select> Define una lista desplegable de opciones. 
<select name="color">
                    <option value="rojo">Rojo</option>
                    <option value="verde">Verde</option>
                </select>
 Se usa junto con <option> para definir las opciones.
<option> Define una opción en una lista desplegable (<select>). 
<option value="rojo">Rojo</option>
 Debe estar dentro de <select>.
<header> Define la cabecera de una sección o página. 
<header>
                    <h1>Bienvenido a mi página</h1>
                </header>
 Útil para agrupar elementos introductorios.
<footer> Define el pie de página de una sección o página. 
<footer>
                    <p>Derechos reservados © 2023</p>
                </footer>
 Comúnmente usado para información de contacto o derechos de autor.
<nav> Define una sección de navegación con enlaces. 
<nav>
                    <a href="/inicio">Inicio</a>
                    <a href="/contacto">Contacto</a>
                </nav>
 Útil para menús de navegación.
<section> Define una sección temática en un documento. 
<section>
                    <h2>Sobre mí</h2>
                    <p>Información sobre mí.</p>
                </section>
 Ayuda a estructurar el contenido de manera semántica.
<article> Define contenido independiente y autónomo, como una publicación o noticia. 
<article>
                    <h2>Título del artículo</h2>
                    <p>Contenido del artículo.</p>
                </article>
 Útil para blogs, foros o noticias.
<aside> Define contenido relacionado indirectamente con el contenido principal. 
<aside>
                    <p>Este es un contenido adicional.</p>
                </aside>
 Comúnmente usado para barras laterales.
<main> Define el contenido principal de un documento. 
<main>
                    <h1>Título principal</h1>
                    <p>Contenido principal.</p>
                </main>
 Debe ser único en la página.
<figure> Define contenido independiente, como imágenes o diagramas. 
<figure>
                    <img src="imagen.jpg" alt="Descripción">
                    <figcaption>Descripción de la imagen</figcaption>
                </figure>
 Se usa junto con <figcaption>.
<figcaption> Define una leyenda para un elemento <figure>. 
<figcaption>Descripción de la imagen</figcaption>
 Debe estar dentro de <figure>.
<time> Define una fecha o hora. 
<time datetime="2023-10-05">5 de octubre de 2023</time>
 El atributo datetime especifica la fecha en formato máquina.
<mark> Resalta texto con un fondo amarillo. 
<p>Este es un <mark>texto resaltado</mark>.</p>
 Útil para enfatizar partes del texto.
<small> Define texto en un tamaño más pequeño. 
<p>Este es un <small>texto pequeño</small>.</p>
 Comúnmente usado para notas legales o derechos de autor.
<strong> Define texto con importancia semántica fuerte. 
<p>Este es un <strong>texto importante</strong>.</p>
 Se muestra en negrita por defecto.
<em> Define texto con énfasis semántico. 
<p>Este es un <em>texto enfatizado</em>.</p>
 Se muestra en cursiva por defecto.
<blockquote> Define una cita larga de otra fuente. 
<blockquote>
                    <p>Esta es una cita larga de otra fuente.</p>
                </blockquote>
 Se usa para citas extensas.
<q> Define una cita corta en línea. 
<p>Él dijo: <q>Esto es una cita corta.</q></p>
 Se muestra entre comillas.
<cite> Define el título de una obra, como un libro o película. 
<p><cite>El Principito</cite> es un libro famoso.</p>
 Se muestra en cursiva por defecto.
<abbr> Define una abreviatura o acrónimo. 
<p>La <abbr title="Organización de las Naciones Unidas">ONU</abbr> es importante.</p>
 El atributo title muestra la descripción completa.
<code> Define un fragmento de código. 
<p>Usa <code>console.log()</code> para imprimir en la consola.</p>
 Se muestra en una fuente monoespaciada.
<pre> Define texto preformateado. 
<pre>
                    Este texto
                    conserva los espacios
                    y saltos de línea.
                </pre>
 Útil para mostrar código o texto con formato específico.
<iframe> Inserta un marco que muestra otra página web. 
<iframe src="https://www.ejemplo.com" width="300" height="200"></iframe>
 Útil para incrustar contenido externo.
<audio> Inserta un archivo de audio. 
<audio controls>
                    <source src="audio.mp3" type="audio/mpeg">
                    Tu navegador no soporta audio.
                </audio>
 El atributo controls muestra los controles de reproducción.
<video> Inserta un archivo de video. 
<video controls width="320" height="240">
                    <source src="video.mp4" type="video/mp4">
                    Tu navegador no soporta video.
                </video>
 El atributo controls muestra los controles de reproducción.
<canvas> Define un área para gráficos renderizados dinámicamente. 
<canvas id="miCanvas" width="200" height="100"></canvas>
 Se usa con JavaScript para dibujar gráficos.
<svg> Define gráficos vectoriales escalables. 
<svg width="100" height="100">
                    <circle cx="50" cy="50" r="40" fill="red" />
                </svg>
 Útil para gráficos y animaciones.
<details> Define un widget de detalles que el usuario puede expandir o contraer. 
<details>
                    <summary>Más información</summary>
                    <p>Este es el contenido adicional.</p>
                </details>
 Útil para secciones colapsables.

                </main>

<map> Define un mapa de imagen con áreas clickeables. 
<img src="imagen.jpg" usemap="#mapa">
                <map name="mapa">
                    <area shape="rect" coords="0,0,50,50" href="enlace1.html">
                    <area shape="circle" coords="100,100,50" href="enlace2.html">
                </map>
 Se usa junto con <area> para definir áreas clickeables.
<area> Define un área clickeable dentro de un mapa de imagen. 
<area shape="rect" coords="0,0,50,50" href="enlace1.html">
 Debe estar dentro de <map>.
<col> Define propiedades de columna en una tabla. 
<colgroup>
                    <col style="background-color: yellow;">
                    <col style="background-color: blue;">
                </colgroup>
 Se usa dentro de <colgroup> para aplicar estilos a columnas.
<colgroup> Agrupa columnas en una tabla para aplicar estilos. 
<colgroup>
                    <col style="background-color: yellow;">
                    <col style="background-color: blue;">
                </colgroup>
 Útil para aplicar estilos a múltiples columnas.
<optgroup> Agrupa opciones en un menú desplegable. 
<select>
                    <optgroup label="Frutas">
                        <option>Manzana</option>
                        <option>Plátano</option>
                    </optgroup>
                    <optgroup label="Verduras">
                        <option>Zanahoria</option>
                        <option>Lechuga</option>
                    </optgroup>
                </select>
 Útil para organizar opciones en grupos.
<keygen> Genera un par de claves para formularios (obsoleto en HTML5). 
<keygen name="clave">
 Ya no se recomienda su uso.
<isindex> Define un campo de búsqueda en una página (obsoleto en HTML5). 
<isindex prompt="Buscar:">
 Ya no se recomienda su uso.
<basefont> Define la fuente base para el documento (obsoleto en HTML5). 
<basefont color="red" size="4">
 Ya no se recomienda su uso.
<applet> Inserta un applet de Java (obsoleto en HTML5). 
<applet code="MiApplet.class" width="200" height="200"></applet>
 Ya no se recomienda su uso.
<acronym> Define un acrónimo (obsoleto en HTML5). 
<acronym title="Organización de las Naciones Unidas">ONU</acronym>
 Reemplazado por <abbr> en HTML5.
<strike> Define texto tachado (obsoleto en HTML5). 
<p>Este es un <strike>texto tachado</strike>.</p>
 Reemplazado por <s> o <del> en HTML5.
<big> Define texto en un tamaño más grande (obsoleto en HTML5).

 Ya no se recomienda su uso.
<center> Centra el contenido horizontalmente (obsoleto en HTML5). 
<center>Este texto está centrado.</center>
 Reemplazado por CSS (text-align: center).

<dir>
                    <li>Archivo1</li>
                    <li>Archivo2</li>
                </dir>
 Reemplazado por <ul> en HTML5.
<font> Define el tipo, tamaño y color de la fuente (obsoleto en HTML5). 
<font color="red" size="4">Este texto es rojo.</font>
 Reemplazado por CSS.
